Para Usuário Comum

Introdução

Este tutorial irá guiá-lo através dos passos necessários para utilizar a tela do portal dos artistas.

Passo 1: Acessar o Portal

Primeiro, acesse o portal dos artistas através do link fornecido.

Login

Faça login com suas credenciais de artista.
Em seguida é necesário selecionar o tipo de login que será feito, como artista ou usuário comum.

Acessar os eventos

Dentro do espaço da tela inicial, é possível visualizar os eventos em destaque da semana,e para ter acesso às informações adicionais do evento é necessário clicar em "Mais Informações"
