Tutorial de Uso para Novos Usuários

Como Acessar Eventos em Destaque

Para acessar os eventos em destaque, siga os passos abaixo:

  1. Vá para a página inicial do Portal dos Artistas.
  2. Na barra de navegação, clique em "Eventos".
  3. Veja a lista de eventos em destaque e clique no evento desejado para mais detalhes.

Como Acessar Editais

Para acessar os editais, siga os passos abaixo:

  1. Vá para a página inicial do Portal dos Artistas.
  2. Na barra de navegação, clique em "Editais".
  3. Veja a lista de editais disponíveis e clique no edital desejado para mais detalhes.

Como Fazer Login

Para fazer login, siga os passos abaixo:

  1. Vá para a página inicial do Portal dos Artistas.
  2. Na barra de navegação, clique em "Login".
  3. Insira seu nome de usuário e senha nos campos apropriados.
  4. Clique no botão "Entrar" para acessar sua conta.
